Help Test the Liberdus App

For Apple iOS use the following link. You will be asked to first install the TestFlight app. Then open TestFlight to install the Liberdus app.
https://testflight.apple.com/join/zSRCWyxy

For Google Android the tester needs to give us their email (gmail preferred).
https://play.google.com/store/apps/details?id=com.jairaj.liberdus

No download, browser option:
https://liberdus.com/test

To migrate existing accounts from browser to app, just do a Backup from the browser to save a file and Restore the file on the app. You can also migrate from app to browser.

Be sure to backup all your accounts periodically so you don't lose any important data. This is a test phase and the network may be reset occasionally.

FAQ

Frequently Asked Questions

Learn more about Liberdus.

What is Liberdus?

+

Liberdus is a decentralized messaging and payment platform that allows users to send secure messages and cryptocurrency payments in a single transaction. It combines the convenience of messaging apps with the power of blockchain technology.

What blockchain does Liberdus use?

+

Liberdus runs on its own chain built using Shardus. Shardus is the protocol layer technology that provides dynamic sharding to solve the blockchain trilemma of scalability, security, and decentralization. Liberdus is built using Shardus to achieve scalability and low transaction fees.

What are the tokenomics for Liberdus?

+

LIB has a fixed supply of 210 million tokens with 18 decimals, following Bitcoin's proven economic model. The token is used for network fees, governance, and incentivizing network participants. No inflation, no dilution.

How can I run a node to earn rewards?

+

Node operators help secure the network and earn LIB rewards. Even during the testnet phase, you can run a validator node and earn real LIB rewards. Check Discord for more details.

How can I add liquidity to earn rewards?

+

Provide liquidity to LIB trading pairs on decentralized exchanges like Uniswap. Liquidity providers earn trading fees and can receive additional LIB rewards.

How do I support the project?

+

You can support Liberdus by contributing code on GitHub, joining our Discord community, providing feedback during testnet, creating educational content, or helping with a variety of other tasks.

Before You Sell

Please read the following important information about LIB tokens.

Polygon Network

The Liberdus token is currently available on the Polygon network. It can be bridged to the Liberdus Mainnet once the Mainnet is launched.

Testnet Tokens

The LIB in the Testnet wallet is not of any value and will not transfer to the Mainnet.

Credit Card Purchase

When the Mainnet has launched, you will be able to buy LIB using a credit card directly.
